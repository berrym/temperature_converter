Temperature Converter

This program is designed to convert degrees Fahrenheit and Celsius.

There are several ways of using it all of them involve passing an option at the command
line.

Valid commands are:

ftoc - convert degrees Fahrenheit to Celsius
ctof - convert degrees Celsius to Fahrenheit
table - print a table of common conversions
help - this text or a list of commands
interactive - an interactive loop to run the individual commands

For example to convert 212 degrees Fahrenheit to Celsius:

cargo run ftoc 212
